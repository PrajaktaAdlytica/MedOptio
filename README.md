# MedOptio Website

Premium SaaS website for MedOptio, a Poland/EU-oriented medication operations
platform for pharmacies, clinics, care homes, and digital health providers.

Live site: https://medoptio.prajakta542277.chatgpt.site

## What is included

- Product-led homepage with MedOptio Review, Refill, and Care storytelling
- Smooth Lenis scrolling and GSAP ScrollTrigger sections
- Sticky workflow narrative, horizontal product section, proof rows, and stacked use cases
- Complete sign-in page with realistic healthcare operations framing
- MedOptio favicon, brand colors, typography, and polished footer

## Local commands

```bash
npm install
npm run dev
npm run build
npm run lint
npm test
```

The site is built with vinext/Next-style App Router components for deployment
through Sites.
